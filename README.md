# Noûs — Landing Page

Uma landing page moderna e responsiva desenvolvida com React e CSS Modules.

## Sobre o Projeto

Landing page criada para apresentar informações de forma clara e atrativa, com design responsivo e interatividade para proporcionar uma boa experiência ao usuário.

## Arquivos

- `src/pages/Home.jsx` — Página principal (une todos os componentes)
- `src/components/Navbar.jsx` — Barra de navegação com scroll
- `src/components/Hero.jsx` — Seção de destaque principal
- `src/components/Features.jsx` — Grade de recursos da plataforma
- `src/components/FeatureCard.jsx` — Card reutilizável de recurso
- `src/components/CTA.jsx` — Seção de chamada para ação
- `src/components/Footer.jsx` — Rodapé da página
- `src/assets/` — Imagens e recursos estáticos
- `README.md` — Documentação do projeto

---

## Tecnologias Utilizadas

- **React** — Componentização e gerenciamento de estado
- **CSS Modules** — Estilização com escopo isolado por componente
- **Vite** — Bundler e servidor de desenvolvimento
- **JavaScript (ES6+)** — Interatividade e animações via IntersectionObserver

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/Luizfelipe075/nous---2026-.git
```

2. Acesse a pasta:
```bash
cd nous-2026
```

3. Instale as dependências:
```bash
npm install
```

4. Rode em modo desenvolvimento:
```bash
npm run dev
```

5. Para gerar o build de produção:
```bash
npm run build
```
